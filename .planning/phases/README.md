# Phase plans

Numbered folders match [`ROADMAP.xml`](../ROADMAP.xml) (e.g. `01-foundation`, `02-product-planning`). Keep **phase-specific** `PLAN.xml` / `SUMMARY.xml` **here**.

- Do **not** put “phase bridge” or roadmap-to-workflow maps under `workflows/` — this directory is the home for phase artifacts.
- [`../workflows/`](../workflows/) holds integration maps, policies, and specs; staff-facing runbooks that render in the app live under **`src/content/docs/`** and are listed in `src/lib/docs/catalog.ts` (`audience: 'admin'`).
