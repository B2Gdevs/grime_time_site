# Method Artifacts SOPs

Each artefact below becomes a checklist + photo guide. Train every role (production, detail, soft-wash) against the steps before they touch the wand.

| Artifact | Purpose | Key Steps | Photo Reference |
| --- | --- | --- | --- |
| Soft Wash Core | Kill organics on wood/siding | 1. Apply diluted chemical evenly 2. Let dwell (don’t dry) 3. Rinse top→bottom 4. Repeat if streaks remain | [Soft-wall spray](#) |
| Controlled Pressure Wash | Remove embedded grime on rough concrete | 1. Hold wand steady at rated distance 2. Sweep with even motion 3. Avoid pausing 4. Edge with lower PSI | [Driveway pass](#) |
| Surface Cleaner Pass | Uniform flatwork cleaning | 1. Align 24″ deck with edge 2. Walk at consistent speed 3. Slight overlap between passes 4. Finish with wand detail | [Surface cleaner diagram](#) |
| Organic Displacement | Remove spider webs on docks/porches | 1. Spray in one direction 2. Don’t spray randomly 3. Expect webs to relocate 4. Re-spray where they reattach 5. Repeat until clear | [Dock web removal](#) |

## Formal Training Sequence

Each crew member sees the same micro-course:

1. **Review artifact checklist** (above).  
2. **Run guided practice**: supervisor verifies two surfaces per artifact.  
3. **Quiz** (questions + troubleshooting).  
4. **Sign-off** (admin marks SOP complete).  

Document completion by tagging `sop-training` in `.planning/STATE.xml` or task 14-06 once ready.
